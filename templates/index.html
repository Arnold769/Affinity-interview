Voucher rules
  • 100 GHS for order value between 1,000 and 5,000 — valid 1 day
  • 500 GHS for order value between 5,000 and 10,000 — valid 5 days
  • 1,000 GHS for order value 10,000 and above — valid 10 days
Customers below 1,000 are not eligible.
Expected headers: Customer ID, Customer First Name, Order Value. Header names are case-insensitive and common variants are supported.
